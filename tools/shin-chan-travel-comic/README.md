# shin-chan-travel-comic — 蠟筆小新風格・2026 旅遊入境新規漫畫

把嚴肅的「2026 全球旅遊入境新規」用《蠟筆小新》風格改編成 5 頁全彩漫畫投影片，
由野原一家（小新、媽媽美冴、爸爸廣志）演出，輕鬆易懂。

## 內容
- **格式**：單一 HTML 檔（`index.html`），內嵌手繪風 SVG 角色，可鍵盤 `← →` 或按鈕翻頁。
- **頁面**：
  1. 出國大計畫！新規大魔王登場（總覽：免簽 ≠ 免申請）
  2. 影響最大的歐盟 ETIAS（+EES 生物辨識）
  3. 英國 UK ETA ＆ 美國 ESTA
  4. 日本 JESTA ＆ 韓國 K-ETA
  5. 專家三大叮嚀（授權綁護照／提前 1 週／只用官方網站）

## 資料來源
內容整理自上傳文件「2026 全球旅遊入境新規說明文件」
（來源：行銷人 MarketersGO、歐盟 ETIAS 官方資訊、英國政府官網）。
各國政策可能調整實施日期，出發前請再次確認各國外交部官網。

## 使用方式
直接用瀏覽器開啟 `index.html` 即可播放。
