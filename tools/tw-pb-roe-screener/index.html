台股 P/B-ROE 合理股價篩選器

全市場迴歸 P/B ~ ROE，找出股價低於模型合理價的上市股票

方法說明

1. 從證交所 OpenAPI 抓取全部上市股票最新一期的「本益比、殖利率、股價淨值比」（BWIBBU_ALL）與最新「收盤價」（STOCK_DAY_ALL）。
2. 用 ROE = 股價淨值比(P/B) ÷ 本益比(P/E) 還原出每檔股票目前的股東權益報酬率（因為 P/B = P/E × ROE 是恆等式）。
3. 對全市場所有股票做「P/B ~ ROE」最小平方法線性迴歸，得到市場對「每一單位 ROE 值多少 P/B」的定價關係：合理P/B = a + b × ROE。
4. 合理股價 = 每股淨值 × 合理P/B（每股淨值 = 收盤價 ÷ 目前P/B）。
5. 篩出「目前股價 < 合理股價」（即目前 P/B 低於迴歸線）的股票，依低估幅度排序。

⚠️ 這是量化篩選工具，不是投資建議。迴歸模型未區分產業（金融股、景氣循環股的合理估值邏輯不同），且僅反映「相對全市場定價」是否便宜，不代表公司基本面沒有風險。虧損（本益比為負/無法計算）的公司會被自動排除，因為此模型無法評估。請自行搭配財報、產業狀況再做投資判斷。
